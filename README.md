## Hello24.ai Assignment Node.js

### Setup
clone this repository

```
$ git clone https://github.com/crudybagger/hello24.git
```

### Install
Run the below command to install the dependencies for WooCommerce and Shopify
```
$ cd hello24 
$ npm install
```

### Run
```
$ node shopify.mjs

$ node woocommerce.mjs
```
